Como a web funciona

Conceitos Web

Os termos por tras de tudo que roda no navegador.

DOM
A estrutura da pagina HTML organizada como uma arvore. JavaScript mexe no DOM pra mudar o que aparece na tela.
Responsivo
Design que se adapta a diferentes tamanhos de tela. Um site responsivo funciona bem no celular e no desktop.
Viewport
A area visivel da tela do usuario. O meta viewport no HTML diz pro navegador como escalar o site.
Breakpoint
O ponto onde o layout muda pra se adaptar a outro tamanho de tela. Tipo 860px pro tablet e 480px pro celular.
CDN
Servidores espalhados pelo mundo que entregam seus arquivos mais rapido. Google Fonts usa CDN.
SSL / HTTPS
O cadeado verde no navegador. Conexao criptografada entre voce e o site. Sem SSL, dados viajam "abertos".
DNS
Traduz nomes de site (google.com) pra enderecos IP. E a agenda telefonica da internet.
Cookie
Um arquivo pequeno que o site guarda no seu navegador pra lembrar de voce. Login, preferencias, carrinho de compras.
Cache
Uma copia local dos arquivos do site. Na proxima visita, carrega mais rapido porque ja tem tudo guardado.
URL / Route
URL e o endereco completo de uma pagina. Route e o pedaco depois do dominio, tipo /pages/basicos.html.
