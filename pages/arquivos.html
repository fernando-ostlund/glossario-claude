Arquivos e estrutura

Tipos de arquivo e conceitos de organizacao que voce encontra nos projetos.

.md
Markdown. Formato de texto simples com formatacao (negrito, titulos, listas). Todas as suas skills sao arquivos .md.
.tsx / .jsx
Arquivos de componentes React. O .tsx e com TypeScript (mais seguro), .jsx e JavaScript puro. E onde a interface do site vive.
.env
Arquivo de segredos (senhas, chaves de API). Nunca deve ir pro GitHub. Sua chave do Gemini ta num .env.
.json
Formato de dados estruturados. Tipo uma planilha, so que em texto. Configuracoes, dados, listas. Tudo pode ser JSON.
node_modules
A pasta onde ficam todas as bibliotecas instaladas. E enorme e nunca vai pro GitHub. O npm install recria ela a partir do package.json.
Diretório / Pasta
Mesma coisa. "Diretorio" e o termo tecnico pra "pasta". ~/ significa sua pasta pessoal.
Path / Caminho
O endereço completo de um arquivo. Exemplo: /Users/fernandoostlund/.claude/skills/. A / separa as pastas.
.gitignore
Arquivo que lista o que o Git deve ignorar. Exemplo: node_modules/, .env, .DS_Store. Evita mandar lixo pro GitHub.
stdout / stderr
Dois canais de saida do terminal. stdout e a resposta normal, stderr e onde os erros aparecem. Parecem iguais, mas sao separados.
Pipe ( | )
Conecta a saida de um comando na entrada de outro. ls | grep .html lista arquivos e filtra so os HTML.
Flag / Argumento
Opcoes extras num comando. Em ls -la, o -la e uma flag que pede mais detalhes. Cada comando tem suas proprias flags.
Binario
Arquivo executavel que o computador roda direto. Nao da pra ler num editor de texto. Programas compilados viram binarios.
Symlink
Um atalho que aponta pra outro arquivo ou pasta. Parece que o arquivo ta ali, mas na real ele ta em outro lugar.
chmod
Comando que muda permissoes de arquivo. Define quem pode ler, escrever ou executar. chmod +x script.sh torna o script executavel.
Processo
Um programa rodando na maquina. Cada aba do navegador e um processo. O terminal onde o Claude roda tambem e um processo.
Exit Code
Numero que um comando retorna quando termina. 0 significa "deu certo". Qualquer outro numero significa que deu algum problema.
