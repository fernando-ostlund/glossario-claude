Arquivos e estrutura

Tipos de arquivo e conceitos de organização que você encontra nos projetos.

.md
Markdown. Formato de texto simples com formatação (negrito, títulos, listas). Todas as suas skills são arquivos .md.
.tsx / .jsx
Arquivos de componentes React. O .tsx é com TypeScript (mais seguro), .jsx é JavaScript puro. É onde a interface do site vive.
.env
Arquivo de segredos (senhas, chaves de API). Nunca deve ir pro GitHub. Sua chave do Gemini ta num .env.
.json
Formato de dados estruturados. Tipo uma planilha, só que em texto. Configurações, dados, listas. Tudo pode ser JSON.
node_modules
A pasta onde ficam todas as bibliotecas instaladas. É enorme e nunca vai pro GitHub. O npm install recria ela a partir do package.json.
Diretório / Pasta
Mesma coisa. "Diretório" é o termo técnico pra "pasta". ~/ significa sua pasta pessoal.
Path / Caminho
O endereço completo de um arquivo. Exemplo: /Users/fernandoostlund/.claude/skills/. A / separa as pastas.
.gitignore
Arquivo que lista o que o Git deve ignorar. Exemplo: node_modules/, .env, .DS_Store. Evita mandar lixo pro GitHub.
stdout / stderr
Dois canais de saída do terminal. stdout é a resposta normal, stderr é onde os erros aparecem. Parecem iguais, mas são separados.
Pipe ( | )
Conecta a saída de um comando na entrada de outro. ls | grep .html lista arquivos e filtra só os HTML.
Flag / Argumento
Opções extras num comando. Em ls -la, o -la é uma flag que pede mais detalhes. Cada comando tem suas próprias flags.
Binário
Arquivo executável que o computador roda direto. Não dá pra ler num editor de texto. Programas compilados viram binários.
Symlink
Um atalho que aponta pra outro arquivo ou pasta. Parece que o arquivo ta ali, mas na real ele ta em outro lugar.
chmod
Comando que muda permissões de arquivo. Define quem pode ler, escrever ou executar. chmod +x script.sh torna o script executável.
Processo
Um programa rodando na máquina. Cada aba do navegador é um processo. O terminal onde o Claude roda também é um processo.
Exit Code
Número que um comando retorna quando termina. 0 significa "deu certo". Qualquer outro número significa que deu algum problema.
