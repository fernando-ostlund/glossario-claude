Conceitos básicos

Os termos que aparecem o tempo todo quando voce usa o Claude Code.

CLI
Command Line Interface. Aquela tela escura onde voce digita comandos. O Claude Code roda ali. Basicamente um chat, so que no terminal.
Terminal
O app onde a CLI roda. No Mac, e o app "Terminal" ou "iTerm2". A janela onde voce conversa com o Claude Code.
IDE
Integrated Development Environment. Um editor de codigo com superpoderes. Exemplos: VS Code, Cursor. Onde os devs escrevem codigo, e onde o Claude Code pode rodar como extensao.
Script
Um arquivo com instrucoes que o computador segue linha por linha. Tipo uma receita de bolo, so que pra maquina. Exemplo: um script Python que analisa um PowerPoint.
Prompt
O texto que voce escreve pra pedir algo pro Claude. Quanto mais claro e especifico, melhor sai.
Modelo / Model
O "cerebro" da IA. Claude Opus e o mais inteligente, Sonnet e rapido e capaz, Haiku e o mais leve. Voce usa Opus agora.
Token
A unidade de medida do texto pra IA. Uma palavra tem mais ou menos 1 a 2 tokens. Quanto mais tokens, mais custa e mais demora.
Contexto
Tudo que o Claude "lembra" durante a conversa. Seus arquivos, instrucoes, historico. Tem um limite (1M tokens no Opus). Quando enche, as mensagens antigas sao comprimidas.
