Conceitos básicos

Os termos que aparecem o tempo todo quando você usa o Claude Code.

CLI
Command Line Interface. Aquela tela escura onde você digita comandos. O Claude Code roda ali. Basicamente um chat, só que no terminal.
Terminal
O app onde a CLI roda. No Mac, é o app "Terminal" ou "iTerm2". A janela onde você conversa com o Claude Code.
IDE
Integrated Development Environment. Um editor de código com superpoderes. Exemplos: VS Code, Cursor. Onde os devs escrevem código, e onde o Claude Code pode rodar como extensão.
Script
Um arquivo com instruções que o computador segue linha por linha. Tipo uma receita de bolo, só que pra máquina. Exemplo: um script Python que analisa um PowerPoint.
Prompt
O texto que você escreve pra pedir algo pro Claude. Quanto mais claro e específico, melhor sai.
Modelo / Model
O "cérebro" da IA. Claude Opus é o mais inteligente, Sonnet é rápido e capaz, Haiku é o mais leve. Você usa Opus agora.
Token
A unidade de medida do texto pra IA. Uma palavra tem mais ou menos 1 a 2 tokens. Quanto mais tokens, mais custa e mais demora.
Contexto
Tudo que o Claude "lembra" durante a conversa. Seus arquivos, instruções, histórico. Tem um limite (1M tokens no Opus). Quando enche, as mensagens antigas são comprimidas.
