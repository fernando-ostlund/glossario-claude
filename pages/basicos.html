Conceitos básicos

Os termos que aparecem o tempo todo quando você usa o Claude Code.

CLI
Command Line Interface. É aquela tela escura onde você digita comandos. O Claude Code roda nela. Pense como um chat, mas no terminal.
Terminal
O app onde a CLI roda. No Mac, é o app "Terminal" ou "iTerm2". É a janela onde você conversa com o Claude Code.
IDE
Integrated Development Environment. Um editor de código com superpoderes. Exemplos: VS Code, Cursor. É onde os devs escrevem código — e onde o Claude Code pode rodar como extensão.
Script
Um arquivo com instruções que o computador segue passo a passo. Como uma receita de bolo, mas para a máquina. Exemplo: um script Python que analisa um PowerPoint.
Prompt
O texto que você escreve para pedir algo ao Claude. Quanto mais claro e específico, melhor o resultado.
Modelo / Model
O "cérebro" da IA. Claude Opus é o mais inteligente, Sonnet é rápido e capaz, Haiku é o mais leve. Você usa Opus agora.
Token
A unidade de medida do texto para IA. Uma palavra tem ~1–2 tokens. Quanto mais tokens, mais custa e mais demora.
Contexto
Tudo que o Claude "lembra" durante a conversa: seus arquivos, instruções, histórico. Tem um limite (1M tokens no Opus). Quando enche, as mensagens antigas são comprimidas.
