Playwright

Status: Ativo

O que faz

Permite que o Claude controle um navegador de verdade. Navegar em sites, clicar, preencher formulários, tirar screenshots, ler o console. Basicamente, dá mãos pro Claude usar o Chrome.

Explica como se eu tivesse 5 anos

Imagina que o Claude tem mãos e pode usar o Chrome. Ele abre um site, clica nos botões, preenche campos, tira foto da tela. É como dar um navegador pro Claude usar igual você usa.

Quando usar

  • Pra testar se seu site funciona
  • Pra extrair informação de um site
  • Pra tirar screenshots automáticos
  • Pra debugar problemas visuais

Quando NAO usar

  • Pra ler arquivos locais (pra isso tem o Read)
  • Pra fazer buscas simples na web (pra isso tem WebSearch)

Tools disponíveis

  • navigate — abre uma URL no navegador
  • click — clica num elemento da página
  • screenshot — tira foto da tela
  • fill_form — preenche campos de formulário
  • evaluate — roda JavaScript na página
  • console — lê mensagens do console do navegador
  • tabs — gerencia abas abertas
  • network_requests — monitora requisições de rede

Exemplo prático

Você acabou de deployar o glossário. Pede pro Claude "abre o site e confere se tá tudo certo". Ele navega, tira screenshot, checa o console, e reporta se tem erro.

> "abre glossario-claudinho.vercel.app e confere se tá ok"

[Playwright] Navigating to glossario-claudinho.vercel.app
[Playwright] Taking screenshot...
[Playwright] Checking console for errors...

Tudo certo. Site carregou em 1.2s, sem erros no console.
