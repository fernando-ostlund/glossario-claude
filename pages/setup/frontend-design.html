frontend-design

/frontend-design

O que faz

Cria interfaces web com alto nível de design. Evita a estética genérica de IA. Gera código HTML/CSS/JS production-grade, tipo coisa que um designer sênior faria.

Explica como se eu tivesse 5 anos

Quando você pede pro Claude fazer um site, essa skill é tipo dar pra ele um olho de designer. Em vez de fazer algo genérico e feio, ele faz algo que parece feito por um profissional. Tipografia cuidada, espaçamento generoso, cores que combinam.

Quando usar

  • Quando quer criar componentes web, páginas ou aplicações visuais
  • Quando o resultado precisa ter qualidade visual alta
  • Quando quer que o código saia bonito de primeira, sem ficar ajustando depois

Quando NAO usar

  • Quando é algo puramente funcional sem interface
  • Quando é backend ou lógica de dados
  • Quando você só quer um script rápido que ninguém vai ver

Exemplo prático

Você pede "cria uma landing page pro meu portfólio". Com essa skill ativa, o resultado sai com tipografia cuidada, espaçamento generoso, cores harmoniosas. Nada daquela cara de template Bootstrap genérico.
