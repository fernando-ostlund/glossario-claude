frontend-design

/frontend-design

O que faz

Cria interfaces web com alto nivel de design. Evita a estetica generica de IA. Gera codigo HTML/CSS/JS production-grade, tipo coisa que um designer senior faria.

Explica como se eu tivesse 5 anos

Quando voce pede pro Claude fazer um site, essa skill e tipo dar pra ele um olho de designer. Em vez de fazer algo generico e feio, ele faz algo que parece feito por um profissional. Tipografia cuidada, espacamento generoso, cores que combinam.

Quando usar

  • Quando quer criar componentes web, paginas ou aplicacoes visuais
  • Quando o resultado precisa ter qualidade visual alta
  • Quando quer que o codigo saia bonito de primeira, sem ficar ajustando depois

Quando NAO usar

  • Quando e algo puramente funcional sem interface
  • Quando e backend ou logica de dados
  • Quando voce so quer um script rapido que ninguem vai ver

Exemplo pratico

Voce pede "cria uma landing page pro meu portfolio". Com essa skill ativa, o resultado sai com tipografia cuidada, espacamento generoso, cores harmoniosas. Nada daquela cara de template Bootstrap generico.
