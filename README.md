# Glossario Claude Code

Base de conhecimento pessoal do Fernando Ostlund para tudo relacionado a Claude Code, IA e desenvolvimento.

**URL:** [glossario-claudinho.vercel.app](https://glossario-claudinho.vercel.app)

---

## O que e este projeto

Um site de referencia pessoal que reune, em um unico lugar, tudo que eu preciso consultar no dia a dia trabalhando com Claude Code. Comecou como um glossario de termos tecnicos traduzidos para linguagem simples, e esta evoluindo para uma base de conhecimento completa.

Eu sou designer, nao sou programador. Este projeto existe porque eu cansei de ficar procurando a mesma informacao em lugares diferentes toda vez que esquecia o que significava um termo ou como usar uma ferramenta. Aqui eu documento tudo do meu jeito, na minha linguagem, para poder voltar e consultar sempre que precisar.

## O que voce encontra aqui

- **Glossario de termos tecnicos** -- CLI, terminal, Git, API, token, deploy e dezenas de outros termos explicados sem jargao
- **Roteiro de cursos** -- os cursos gratuitos da Anthropic Academy organizados na ordem certa para quem esta comecando
- **Dashboard do meu setup** -- inventario completo de tudo que tenho instalado no Claude Code: 48 skills, 12 agentes, 1 hook de seguranca e 4 conexoes MCP. Cada item mostra o que faz, quando usar e qual comando aciona
- **E o que vier pela frente** -- workflows pessoais, links de ferramentas, videos do YouTube, processos documentados

## Para quem serve

Para mim, principalmente. Mas tambem pode servir para qualquer designer ou profissional nao-tecnico que esta aprendendo a usar Claude Code e precisa de um guia em portugues, sem complicacao.

## Stack

HTML + CSS + JS puro. Sem framework, sem build, sem dependencias. Abre no browser, funciona. Deploy automatico na Vercel.

## Como contribuir

Este e um projeto pessoal em constante evolucao. Se voce chegou aqui e quer sugerir um termo ou correcao, abre uma issue.
